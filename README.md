# Library Management API

Spring Boot ile geliştirilmiş basit bir Library Management REST API.

## Features

- Add Book
- List Books
- Delete Book
- Borrow Book

## Technologies

- Java
- Spring Boot
- Spring Data JPA
- PostgreSQL
- DTO Pattern
- Validation

## API Endpoints

POST /books  
GET /books  
DELETE /books/{id}

## Run Project

```bash
mvn spring-boot:run
